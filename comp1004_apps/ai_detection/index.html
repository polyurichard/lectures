AI vs Human Writing Classifier

Data Points

  • Each point = one paper
  • Red points: AI-generated papers
  • Blue points: Human-written papers
  • Y-axis: Number of GenAI words

Classification Logic

  • Above threshold → Classified as AI
  • Below threshold → Classified as Human

Shaded Areas

  • Light red: AI classification zone
  • Light blue: Human classification zone

Classification Results

  • Correct: Red in red area, blue in blue area
  • Incorrect: Red in blue area, blue in red area

Interactive Features

  • Black line = adjustable threshold
  • Move slider to see classification changes
